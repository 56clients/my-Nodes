##HTML介绍

###HTML与CSS的关系
* css的代码可以插入html,插入语法为`<style type="text/css">css-code</style>`

###HTML标签
* 标题标签 `<h1>content</h1>`
* 段落标签 `<p></p>`
* 图片标签 `<img src="img-link"></img>`

###HTML文件基本结构
html的固定结构
```html
<html>
    <head>...</head>
    <body>...</body>
</html>
```
* 根标签 `<html></html>`
    * 所有网页标签全部位于根标签
* 定义头部的标签 `<head> </head>`
    * 所有头部元素的容器
    * 头部元素主要有 `title` `script` `style` `link` `meta`
* 主体标签 `<body></body>`
    * 该标签之间的内容是网页的主要内容,会在浏览器中显示
