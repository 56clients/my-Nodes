## 在构造器中调用构造器
> 为了避免重复写代码，可以在一个类中定义若干个构造器，在一个构造器中调用另一个构造器

```java
class Flower{
    private int val = 0;
    private String name = "init";
    Flower(){
    }
    Flower(int val){
        this.val = val;
    }
    Flower(String name){
        this.name = name;
    }
    Flower(int val,String name){
        this(val);
        this.name = name;
    }
    //error
    //一个构造器最多可调用自身其他构造器一次
    Flower(int val,String name){
        this(val);
        this(name);
    }
}
```
**注：** 一个构造器最多可调用自身其他构造器一次
